## 🚀 Overview

This portfolio highlights my journey from a Laravel-focused intern to a Senior Frontend Architect and QA specialist. It features high-impact projects built for **PT Admin Pintar Kita**, **PT Permata Azzuri Sehat**, and more.

## 🛠 Tech Stack

- **Frontend**: Next.js 14, React 18, Tailwind CSS, Shadcn UI
- **3D Graphics**: Three.js, React Three Fiber, Drei
- **State & Communication**: Zustand, Axios, Socket.io (WebSockets)
- **Animation**: Framer Motion, React Tilt
- **Testing**: Playwright (E2E), Manual QA

## 🌟 Key Projects

- **Unified Dashboard Mimin.io**: Centralized enterprise hub with low-code workflow builders.
- **PT Permata Azzuri Sehat RME**: Electronic Medical Record system built with Next.js.
- **WhatsApp Official Campaign**: Enterprise blast and campaign management dashboard.
- **Flix Cinema**: Premium cinema portal with booking capabilities.
- **Beautica KPI Dashboard**: Sales performance and growth tracking analytics.
